# 🌃 Surveillance 👁️

![GIF](preview1.gif) 

---

**Surveillance**는 CCTV를 통해 몬스터를 감시하며 야간 근무 동안 발생하는 다양한 이상현상을 해결하고 생존하는 **2D 공포 게임**입니다. 플레이어는 제한된 정보를 바탕으로 신속하고 전략적으로 행동하여 몬스터의 상태를 관리하고 생존 해야 합니다.
